Para executar o projeto abra a pasta fleet-control-web abra o terminal na pasta e execute no terminal npm run dev (Necessário node instalado)
e para executar a api abra a pasta fleet-control-api\src\FleetControl.Api abra o terminal na pasta e execute no terminal dotnet run(Necessario .net10 instalado)
abra no navegador http://localhost:5173/veiculos
